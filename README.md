# Animated Portfolio Website 🎨

A modern, highly interactive, and visually stunning portfolio website inspired by Cynthia Ugwu. This project demonstrates advanced web animation techniques, smooth scrolling, and a premium design aesthetic using vanilla web technologies.

## 🌟 Features

- **Smooth Scrolling**: Implemented using [Locomotive Scroll](https://locomotivemtl.github.io/locomotive-scroll/) for a fluid, premium browsing experience.
- **Advanced Animations**: Powered by [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/) for intricate entry animations, hover effects, and custom cursors.
- **Custom Cursor**: A dynamic custom cursor that interacts with different elements on the page.
- **Modern Typography & Layout**: Clean, minimalist design focusing on typography and large imagery, achieving a high-end look.
- **Responsive Design**: Built to look great across different screen sizes.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and layout.
- **CSS3**: Custom styling, flexbox/grid layouts, and responsive media queries.
- **JavaScript (ES6+)**: DOM manipulation and animation logic.
- **GSAP**: For complex, timeline-based animations.
- **Locomotive Scroll**: For smooth, momentum-based scrolling.
- **Remix Icons**: For crisp, scalable icons.

## 🚀 Getting Started

### Prerequisites

No complex build tools or environments are required. All you need is a modern web browser.

### Installation & Execution

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd Animated-Frontend-webpage/Main-Files
   ```
3. Open `index.html` in your preferred web browser. Alternatively, you can use a local development server like VS Code's "Live Server" extension for the best experience.

## 💡 Highlights

This project serves as an excellent showcase of how to combine modern JavaScript animation libraries (GSAP) with smooth scrolling utilities (Locomotive Scroll) to create award-winning web experiences without relying on heavy frameworks. It demonstrates how to orchestrate complex timelines and manage scroll-triggered animations.
